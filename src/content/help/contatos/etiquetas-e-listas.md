São duas formas de agrupar gente, com propósitos diferentes. Confundir as duas é comum — vale um minuto para entender.

## Etiquetas: características

Etiqueta é uma marca colada no contato: `Cliente`, `Orçamento enviado`, `VIP`, `Não responde`. Um contato pode ter várias.

**Criar:** Configurações → Tags. Nome e cor.

**Aplicar:** na conversa, abra **Dados do contato** e use a seção de etiquetas. Também dá para aplicar em massa durante a [importação](/ajuda/contatos/cadastrar-e-importar).

**Onde elas trabalham para você:**

- **Filtrar o funil** do [CRM](/ajuda/crm/usando-o-funil).
- **Mirar campanhas** — "enviar para todo mundo com a etiqueta Orçamento enviado".
- **Disparar fluxo automático** — existe um gatilho *Tag adicionada*: ao marcar alguém com uma etiqueta, um [fluxo](/ajuda/fluxos/gatilhos-do-fluxo) começa sozinho. Ótimo para "marcou como Orçamento enviado → manda a proposta e agenda retorno".
- **Aumentar a pontuação** do lead, se você usa [pontuação de leads](/ajuda/crm/pontuacao-de-leads).

## Listas: recortes fixos

Lista é um conjunto fechado de contatos, montado para uma finalidade: `Feira de setembro`, `Alunos turma 3`, `Importação 12/03`.

Diferença prática:

| | Etiqueta | Lista |
|---|---|---|
| Serve para | Qualificar o contato | Agrupar para um envio |
| Muda com o tempo | Sim, é o estado atual | Não, é um recorte |
| Aciona automação | Sim (gatilho de fluxo) | Não |
| Usada em campanha | Sim | Sim |

Listas são criadas na **importação** ou na hora de montar a campanha.

Regra simples: **etiqueta descreve o que a pessoa é; lista guarda de onde ela veio.**

## Como usar bem

- **Poucas etiquetas, com nome claro.** Vinte etiquetas parecidas ninguém mantém. Comece com quatro ou cinco.
- **Padronize entre a equipe** — combine se é `Cliente` ou `Clientes`, senão viram duas.
- **Etiquete no fim do atendimento**, enquanto o contexto está fresco. Depois ninguém volta.
- **Use as etiquetas para segmentar campanha**, não a base inteira. Mensagem relevante para pouca gente rende mais e arrisca menos o número do que mensagem genérica para todo mundo.

## Onde ver quem tem cada etiqueta

Na tela de Contatos você filtra pela etiqueta e vê quantas pessoas estão marcadas. No funil do CRM, a etiqueta aparece no rosto do cartão — dá para bater o olho e saber com quem está falando.
